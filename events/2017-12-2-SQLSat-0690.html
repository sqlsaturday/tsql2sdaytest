SQLSaturday #690 - Lima 2017
This event was held on 2017-12-2

Event Location:

ISIL Campus San Isidro

Av. Sanchez Carrión 285 (Ex Av. Pershing)

Lima, Lima, Peru

Schedule

PDF of Schedule

Azure Data Lake vs Azure HDInsight

Abstract:

Ventajas y desventajas entre éstas dos herramientas que nos ofrece la nube para poder aplicar el concepto de Big Data basado sobre la plataforma Hadoop.

 

Speaker(s): Jorge Pastor,

De 0 a 100 con Reporting Services 2016

Abstract:

En esta sesión se hará uso de la herramienta de reporting services de SQL Server 2016. A su vez se demostrarán las nuevas características evolución, entre otros.

 

Speaker(s): Julio Enrique Urriola Barra,

Administración de SQL Server 2017 desde Linux

Abstract:

Track orientado al Despliegue, Gestión de Seguridad, In Memory OLTP y Always On de SQL Server 2017 desde el Sistema Operativo Linux

 

Speaker(s): German Cayo,

Los negocios de hoy en dia con PowerApps, Power BI y Microsoft Fl

Abstract:

En esta sesión vamos a ver como crear aplicación móviles de manera rápida para usuario de negocio apoyándonos con PowerApps y Microsoft, así mismo generar reportes poderosos con Power BI.

 

Speaker(s): Juan Rafael,

Ciencia de Datos con R y Microsoft Power BI

Abstract:

Ciencia de Datos con R y Microsoft Power BI

 

Speaker(s): Alejandro Humberto Sánchez Núñez,

La ultima milla...Power BI Report Server - OnPremise

Abstract:

En esta sesión veremos los aspectos relevante de Power BI On Premise (en tierra). -Instalación de la herramienta Servidor y Cliente -Configuración del portal -Administración de portal y seguridad -Beneficios y Restricciones

 

Speaker(s): Vladimir Guzman,

Data Migrations using SQL technologies

Abstract:

En esta sesion hablaremos sobre como utilizar SSIS, .NET and MSFT SQL para lograr mover informacion de un sistem a otro. Ya sea un ERP migration o cargar informacion a una Data Warehouse.

 

Speaker(s): Eduardo Favio Angeles,

PowerBI y Microsoft Dynamics 365

Abstract:

En esta sesión vamos a mostrar como podemos dar poder a los usuarios de negocio informes poderosos con información de CRM

 

Speaker(s): Juan Rafael,

Pariticionamiento y Compresion de Datos en SQL Server

Abstract:

en esta sesión veremos e implementaremos el particionamiento en tablas SQL Server (Creación de Esquema, Creación de función, Creación de tabla). Además el uso y ventajas de la compresión de datos (filas y paginas)

 

Speaker(s): Vladimir Guzman,

Visualizaciones con R en PowerBI.

Abstract:

Se dará los conceptos de R. R en la Arquitectura de Microsoft. Como usar R en PowerBI. Utilización de paquetes de visualización como ggplot2, lattice, etc.

 

Speaker(s): Jorge Michael Muchaypiña Gutierrez,

Optimización de SQL Server para IOT

Abstract:

Preparando SQL Server para trabajar con información de múltiples mediciones de sensores IOT

 

Speaker(s): Marlon Leandro,

BI360 CPM Suite al rescate de las necesidades de Financial Reporting Planning

Abstract:

La oficina de finanzas y contabilidad de las organizaciones tienen necesidades específicas de gestión de la información, así como de soporte a sus procesos financieros de budgeting, forecasting y consolidación financiera para la toma de decisiones. Desde reporting y planning financiero y operacional, hasta contar con una data warehouse flexible y potente, BI360 CPM Suite es una solución completa construida sobre plataforma Microsoft BI que sale al rescate de estas y otras necesidad de las organizaciones.

 

Speaker(s): Brian Koo,

Nos vamos a la nube! cómo llevar tu SQL Server al servicio de Azure

Abstract:

La nube presenta muchos beneficios pero las incógnitas flotan en el aire. ¿cómo lo hago? ¿qué implica? ¿cuanto me cuesta? En esta sesión exploraremos las diferentes opciones, los retos, las herramientas. Todo lo que debes saber para decidir con certeza que la nube es el camino (y sustentarlo a tus jefes)

 

Speaker(s): Alberto De Rossi,

Analysis Services Tabular Model

Abstract:

- Se creara un cubo tabular con SQL Server Data Tools - Se migrara un Powerpivot a un modelo Tabular - Manejar diferentes orígenes de Datos - Se manejara jerarquías - Creación de Particiones - Permisos - Medidas Calculadas con DAX - Como explotar nuestro cubo con Excel, SSRS y PowerBI.

 

Speaker(s): Jorge Michael Muchaypiña Gutierrez,

El despertar de la fuerza con Power BI

Abstract:

En esta sesión se abordarán los diferentes modos de conexión permitidos de Power BI , así como el uso de los gateways para el auto refresco de los datos.

 

Speaker(s): Raul Martin Sarachaga Diaz,

Hybrid BI con Power BI Report Server y PowerBI.com

Abstract:

El concepto de la charla es guiar de porqué existe PBI RS y mostrar la diferencia con powerbi.com (servicio nube); ademas de por que usar PBI RS, cuando es recomendable ir a por esta solución y como sacarle a las caracteristicas que van siendo habilitadas para esta versión. Punto aparte no olvidar que si se usa PBI RS y quieres ir al servicio, puedes hacer cuando se necesite o la organización lo requiera.

 

Speaker(s): Alejandro Humberto Sánchez Núñez,

Office y Power BI, usando Visio y Power Point para analizar y presentar datos

Abstract:

Power BI evoluciona constantemente y Microsoft se está esforzando por integrarla con sus productos. En esta sesión veremos como puedes crear diseños personalizados en Visio y usarlos en Power BI para analizar datos. Además, usaremos Power Point para presentar los tableros de forma interactiva. Olvídate de los gráficos estáticos, ahora puedes tener toda la interactividad de Power BI en tu Power Point (Power x Power = Super Power!)

 

Speaker(s): Alberto De Rossi,

Explorando los sabores de SQL Azure Data Warehouse

Abstract:

Explorará las características principales de este servicio en la nube, así como podemos migrar la data on premises a la nube utilizando servicios como Polybase, Azure Data Factory e Integration Services.

 

Speaker(s): Raul Martin Sarachaga Diaz,

Streaming de datos con Power BI

Abstract:

Aprenda a mostrar sus datos en tiempo real a través de Power BI. Analice los datos que envían los sensores IoT

 

Speaker(s): Marlon Leandro,

Artificial Intelligence in SQL Server 2017

Abstract:

Se presentara en esta sesión la integracion de la inteligencia artificial con SQL Server 2017.

 

Speaker(s): Nicolas Nakasone,

Dashboards con Mobile Reports

Abstract:

Se aprenderá como construir Dashboards con Mobile Reports y la forma de consumo. Se verán aspectos como: uso de filtros y tipos de gráficos tales como Gauges, Charts, Datos Geográficos, Mapas, entre otros.

 

Speaker(s): Julio Enrique Urriola Barra,
